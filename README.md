# scilab-cli-hello-world

## Description
A POC for scilab programming language.

## Tech stack
- java
- scilab
- python

## Docker stack
- ubuntu:22.04
- rzuckerm/scilab

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
